# 차량의 외장 패널을 사용하여 침수심 계산

## 차량의 외장 패널
- 바퀴
- 사이드미러
- 범퍼
- 등등

## 현재 로직
```
차량 감지(YOLO11n)
    │
    ▼
차량 바퀴, 사이드미러, 범퍼 등등의 높이 파악(문헌조사) - 완
    │
    ▼
차량의 외장 패널 분할(Segmentation) - 완
    │
    ▼
픽셀/cm 비율 계산
    │
    ▼
수면선 픽셀 위치 감지
    │
    ▼
실제 수위 cm 출력!
```

## 추가로 할 수 있는거
- 차량의 종류를 학습시킨다.
```

차량 종류 분류(classification)
    │
    ▼
차량 마다 바퀴, 사이드미러, 범퍼 등등의 높이 파악(문헌조사)
    │
    ▼
차량의 외장 패널 분할(Segmentation) - 완
    │
    ▼
픽셀/cm 비율 계산
    │
    ▼
수면선 픽셀 위치 감지
    │
    ▼
실제 수위 cm 출력!
```
---

## 폴더 구조
```
Car_detection -> 차량 감지

Car_classification -> 차량 종류 분류

Car_part_segmentation -> 차량의 외장 패널 분할
```
